# H1D024120-PraktikumKB-Pertemuan10

NIM = H1D024120
2 digit terakhir NIM = 20
digit baru = 20

Langkah
Seleksi : digit pertama dari digit baru = 2
Crossover : digit kedua dari digit baru = 0
Mutasi : jumlah digit I dan digit II dari digit baru = 2 + 0 = 2

Metode
Seleksi : RWS /  Roulette Wheel Selection
Crossover : One Point
Mutasi : Uniform
